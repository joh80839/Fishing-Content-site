### Top 3 Beginner Rod and Reel Combos Under $50

## Purpose & Why

This website describes three rod and reel combos that are under $50 for beginners. Rod and reel combos are considered the most beginner friendly tools to catch fish with. Since combos under $50 is considered non-expensive in today's market, this site will help people who are new to fishing and have a low budget to spend on fishing equipment find a decent rod and reel combo.


### Tools Used

Visual Studio was used to create the HTML, CSS, JavaScript, and Markdown documents for the website. GitHub Desktop was used to send project changes to a GitHub repository. These tools were chosen based on the experience level the developer has with them and how useful they are when creating web applications. ChatGPT was used to aid in the creation of some features in JavaScript and CSS.

## What's Changed?

Project 01 had a main page that included a title with a short moving line feature that traveled beneath it, and a top blue navigation bar with words "Home" and "About" that linked to their corresponding webpages. The "About" page leads to an unfinished page. The center of the main page included three pictures of rod and reel combos with interactive hover features that caused the pictures to expand and reveal their product names. These pictures were linked to webpages that said: "To be Continued". 

Project 02 includes more updates from Project 01. Project 02 has a finished main page. The "About" page leads to a finished page with information about the developer. The three rod and reel combo pictures have hover interactivity and moving lines underneath them to attract the user to interract with them. There are double tap features included on the pictures so users on mobile can tap once to view the product names and tap again to view the webpages for more details about the products. 

The rod and reel pages are finished and include more pictures and information about the products. Each page includes a reference section and a "Back to Home" button, so users don't have to scroll up to return to the main/home page. If users choose to use the browser's return arrow to go back to the main page, JavaScript features were put in place to allow the pictures to return to their normal size.
